Product Principles

PM Thinking and PM Interview Prep

What it is: A reading list and cheat sheets covering product strategy, design, execution, and behavioral frameworks — with worked examples modeled on real FAANG PM interview loops (design, execution, leadership).

Who it helps: Aspiring PMs preparing for their first role, current PMs prepping for Meta / Google / Amazon interviews, engineers and designers making the jump into product.

Start PM Prep

Coding for Product Leaders

Coding 101: HTML / CSS / JS

What it is: A project-based intro to front-end web development — build real websites with HTML, CSS, JavaScript, and jQuery, following slides, capstone projects (Sally Says, Cash Register, Rock-Paper-Scissors), and answer keys.

Who it helps: Non-technical PMs and founders who want to ship their own landing pages, career switchers becoming front-end developers, and anyone who wants to speak engineering fluently.

Start Coding 101

Production Code Setup

Mac, VS Code, CLI, Github Cheat Sheet

What it is: The environment cheat sheet — configure a Mac for development, install and tune VS Code, survive the terminal, and master day-to-day Git and GitHub workflows (branches, PRs, merges, conflicts).

Who it helps: New engineers setting up a laptop on day one, PMs who want to pull down a repo and run it locally, and anyone tired of Googling the same git commands every week.

Open the Cheat Sheet

Python 101

Data Types and Functions

What it is: A single-page cheat sheet covering Python fundamentals — variables, data types, lists / dicts / tuples, control flow, and functions — optimized for fast review rather than long lectures.

Who it helps: PMs and analysts who need to read or write Python scripts, data folks returning to Python after time away, and students who just want the essentials before an interview or first project.

Open Python 101

SQL 101

SQL and Google StratOps Interviews

What it is: The SQL essentials — SELECT, joins, aggregations, window functions — plus interview-style drills modeled on Google's Strategy & Operations loop, so practice mirrors the real thing.

Who it helps: Candidates prepping for StratOps / BizOps / Data Analyst interviews, PMs who want to pull their own metrics instead of waiting on a dashboard, and engineers brushing up on query fundamentals.

Open SQL 101

How to use these materials

Pick your path — PM, Engineer, or Analyst

Product Manager Path
  • Start with Product Principles for frameworks and interview prep.
  • Layer on Coding for Product Leaders to build empathy with your engineers.
  • Keep SQL 101 open for self-serve metrics.
Engineer Path
  • Start with Production Code Setup on day one of a new laptop.
  • Use Coding 101 if you're new to front-end work.
  • Bookmark Python 101 and SQL 101 as quick references.
Analyst / StratOps Path
  • Lead with SQL 101 — it was built for this interview loop.
  • Use Python 101 for quick scripting and data cleanup.
  • Review Product Principles for cross-functional strategy questions.
